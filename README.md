## üìö About Configurations and YAML

Configurations tell software how to behave. You can change them to suit your needs. YAML is a common format for storing these settings since it is both readable and flexible.

In YAML files, you will see keys and values arranged with spaces and indentation. For example, a server setting might look like this:

```yaml
server:
  port: 25565
  maxPlayers: 20
```

Moxie helps you view and modify such keys without messing up the format.
